1 Qual o adorno desta vida? É o amor. } Bis
Alegria é concedida pelo amor. } Bis
É benigno, é paciente,
Não se torna maldizente, } Bis
Este meigo amor.
2 Com suspeitas não se alcança, doce amor } Bis
Onde houver desconfiança, ai do amor. } Bis
Pois mostremos tolerância;
Muitas vezes a arrogância, } Bis
Murcha e mata o amor.
3 ‘Inda quando for custoso, nutre amor. } Bis
Ao irado e mui furioso mostra amor. } Bis
Não te dês por insultado,
Mas responde com agrado, } Bis
Vence pelo amor.
4 Não te irrites, mas tolera com amor. } Bis
Tudo sofre, tudo espera pelo amor. } Bis
Desavenças e rancores
Não convém a pecadores, } Bis
Salvos pelo amor.
5 Pois irmão ao teu vizinho mostra amor. } Bis
O valor não é mesquinho deste amor. } Bis
O supremo Deus nos ama,
Cristo para os céus nos chama, } Bis
Onde reina o amor. - S.P.K.
